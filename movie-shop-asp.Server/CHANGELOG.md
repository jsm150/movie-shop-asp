이 파일은 Visual Studio가 프로젝트를 생성한 방법을 설명합니다.

이 프로젝트를 생성하는 데 다음 단계를 사용했습니다.
- 새 ASP\.NET Core Web API 프로젝트를 만듭니다.
- 프런트 엔드 프로젝트에 대한 참조를 추가하고 SPA 속성을 설정하려면 프로젝트 파일을 업데이트하세요.
- SPA 프록시를 시작 어셈블리로 등록하려면 `launchSettings.json`을(를) 업데이트하세요.
- Docker 빌드를 설정하려면 `dockerfile`을(를) 추가하세요.
- 시작 프로젝트 목록에 프로젝트를 추가합니다.
- 이 파일을 씁니다.
